A few years ago I had realized how many different financial account I would need to log into in order to check my financial standing. During my free time I decided to take a shot at finding a solution to this by bringing all of my financial account data into one place.

To accomplish this I began researching different API's that were able to link into many difference financial institutions. The best solution that I found was the Plaid framework. After deciding on the framework, I built a web application that would allow to get all of the data aggeregated into one location.

I created a web application using Node.js, a database to store user's credentials, accounts, and data using Mongo DB, and a cache to handle logged in users as well as a non persisting way to display their financial data using Redis. By bringing this all together I was able to create a viable product that aggregated one's financial data into one convinient location using a modern stack of technology, learning and building each system from scratch.
